## Project Description
Odin Recipes is my first web development project built using **HTML**.  
The goal of this project is to practice the basics of HTML structure, links, lists, images, and multiple webpages.

This project is part of my learning journey through **The Odin Project**.

## Features
- A homepage that lists different recipes  
- Recipe images displayed on the homepage  
- Clickable links that navigate to individual recipe pages  
- Separate pages for each recipe with ingredients and instructions  

## Technologies Used
- HTML5

## What I Learned
- How to structure a webpage using semantic HTML  
- How to create links between multiple pages  
- How to use lists (`ul`, `ol`) for content organization  
- How to add and display images on a webpage  
- How to organize files and folders for a small project
